# SentinelDrive: Emergency Response Ecosystem

### 🏆 Hackathon Vertical: NHAI / Smart City Infrastructure

**Problem:** High latency and manual communication in emergency dispatching.
**Solution:** A unified agentic ecosystem connecting Victims, Dispatchers, and Responders.
**Agentic Logic:** Built using Google Antigravity to handle multi-scene state transitions from incident detection to 3D-navigation routing.
**Tech Stack:** React, Tailwind CSS, Google Maps SDK Logic, Leaflet.
